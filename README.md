# Engineering Exploration

This repository documents my six-month exploration of robotics, embedded systems, intelligent machines, and systems engineering.

The purpose is not only to collect notes, but to build a personal engineering knowledge base through projects, design decisions, test results, and lessons learned.

## Main Goals

- Explore robotics and embedded systems as a career direction.
- Build practical experience with motors, sensors, microcontrollers, and control logic.
- Practice systems engineering through requirements, architecture, risks, and test plans.
- Connect this exploration to long-term projects such as Orion and Kairo.

## Repository Structure

- `00-Home/` — planning, journal, goals, and baseline notes
- `01-Concepts/` — technical concept notes
- `02-Projects/` — project documentation and implementation notes
- `03-Architectures/` — system designs and diagrams
- `04-LessonsLearned/` — mistakes, fixes, and engineering takeaways
- `05-Resources/` — useful books, videos, articles, and references
- `06-WeeklyReviews/` — weekly progress reviews
